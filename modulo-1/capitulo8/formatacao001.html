Principais formatações

Negrito / Destaque

Negrito

Nesta frase, temos um texto em negrito usando a tag <b>.

O elemento HTML <b> define um texto em negrito, sem nenhuma importância extra.

Destaque

Nesta frase, temos um texto em destaque usando a tag <strong>

O elemento HTML <strong> define um texto com grande importância. O conteúdo interno normalmente é exibido em negrito.

Itálico / Ênfase

Itálico

Nesta frase, temos um texto em itálico. Usando a tag <i>

O elemento HTML <i> define uma parte do texto em uma voz ou modo alternativo. O conteudo normalmente é exibido em italico

Dica: A tag <i> é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc...

Ênfase

Nesta frase, temos um texto em ênfase, usando a tag <em>

O elemento HTML <em> define o texto enfatizado. O conteudo interno normalmente é exibido em itálico.

Dica: Um leitor de tela pronunciará as palavras <em> com ênfase, usando a tonicidade verbal.
