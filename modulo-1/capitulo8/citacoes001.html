Citações

Neste capítulo, veremos os elementos <blockquote>, <q>, <abbr>, <address>, <cite>, e <bdo>.

Elementos de citações HTML

Citações simples

A tag HTML <q> define uma citação curta.

Os navegadores normalmente inserem aspas ao redor da citação:

Como diria o Gustavo Guanabara: O computador é um burro muito rápido.

O objetivo do WWF's é construir um futuro onde as pessoas vivam em harmonia com a natureza.

Citações completas

HTML <blockquote> para citações

Segundo Gustavo Guanabara, no seu curso de HTML e CSS:

Uma blockQuote serve para fazer uma citação/referência a algo, uma citação de bloco, uma citação mais longa.
Esta frase é referente ao link da tag cite.

O elemento HTML <blockquote> define uma seção que é citada de outra fonte.

Os navegadores geralmente recuam elementos <blockquote>.

HTML <cite> para titulo do trabalho

A tag HTML <cite> define o titulo de um trabalho criativo (Exemplo: um livro, um poema, uma musica, um filme, uma pintura, uma escultura, etc.)

Abreviações - Siglas

HTML <abbr> para abreviações

A tag HTML <abbr> define uma abreviação ou um acrônimo, como: HTML, CSS, JS, ASAP, ATM.

Marcar abreviações pode fornecer informações úteis para navegadores, sistemas de tradução e mecanismo de busca.

DICA: Use o atributo global title para mostrar a descrição da abreviação/sigla quando você passar o mouse sobre o elemento.

Exemplo:


                <p>A <abbr title="World Healt Organization">WHO</abbr>foi fundada em 1948.</p>

Texto invertido

HTML <bdo> para substituição bidirecional

BDO significa Substituição Bidirecional.

A tag HTML <bdo>é usada para substituir a direção atual do texto:

Estou aprendendo a criar coisas em HTML.
Este texto sera escrito da direita para esquerda.
