Centralizando

Como centralizar uma div ou qualquer coisa com CSS

Todas as imagens tem como display: inline, temos que usar o display: block

Agora vamos centralizar uma outra imagem na vertical e outra na horizontal de jeitos diferentes.

Para mudar a primeira imagem vamos usar o position para absolute, ou seja, nos mesmo vamos dar o posicionamento que queremos para a imagem.

EXEMPLO:

Como vimos, se usarmos apenas o top: 50% a imagem vai ficar centralizada abaixo da linha desejada, que seria no meio a esquerda. Para resolver isso, temos o transform com os valores translateX(horizontal) e translateY(vertical)
